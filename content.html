Multiple lists can be joined using the + operator to form a new list::

Basic List Concatenation

list1 = [1, 2, 3] list2 = [4, 5, 6] combined = list1 + list2 print(combined)

Multiple Concatenations

You can chain multiple concatenations:

a = [1, 2] b = [3, 4] c = [5, 6] result = a + b + c print(result)

Concatenation Creates a New List

Concatenation creates a new list without modifying the originals:

original1 = [1, 2] original2 = [3, 4] combined = original1 + original2 print(original1) print(original2) print(combined)

Can Only Concatenate Lists to Lists

You can only use + with two lists. Trying to concatenate a list with another type causes a TypeError as the operands are incompatible:

my_list = [1, 2] print(my_list + 3) print(my_list + [3])
